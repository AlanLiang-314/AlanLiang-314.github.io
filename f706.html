[題解]f706: 時區 (Zone)

Posted on Sat, Sep 4, 2021 題解 python
f706. 時區 (Zone) - 高中生程式解題系統

https://zerojudge.tw/ShowProblem?problemid=f706

def zero(a):
    if a < 10:
        return "0"+str(a)
    else:
        return a

H,M,S,T = map(int, input().split())
skip = 90*T
M += skip%60
H += skip//60+M//60
M=M%60
H=H%36
print("{}:{}:{}".format(H,zero(M),zero(S)))
