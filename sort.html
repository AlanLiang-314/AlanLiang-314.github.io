[C++] algorithm sort

Posted on Tue, Aug 24, 2021 C++

sort 可以針對array或vector進行排列

array

int a[10]={3,1,4,1,5,9,2,6,7,8,0};
sort(a,a+9);
//or
sort(a,a+sizeof(a)/sizeof(int));

vector

vector<int> v = {3,1,4,1,5,9,2,6,7,8,0};
sort(v.begin(),v.end())

使用sort(v.rbegin(),v.rend())可以在排序後反轉，和使用sort後再reverse類似。
