[C++] string

Posted on Mon, Apr 5, 2021 C++
實作日期：20210405(補做)

宣告

  • string str1宣告一個空的string
  • string str2("word")宣告一個內容為word的string

方法

  • str.size()求str長度
  • str.empty()求str是否長度為0，如為0回傳1
  • str1 + str2將str1和str2連接

和char轉換

string可以轉換成char，但char不能轉換成string
